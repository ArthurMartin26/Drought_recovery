Elevation data: Terrain elevation data were sourced from the Shuttle Radar Topography Mission (SRTM) Global Land Digital Elevation Model (GL3, ~90 m resolution) via the OpenTopography API and downloaded as a GeoTIFF using a WGS84 bounding box covering Nigeria. The downloaded raster was validated by checking its spatial extent, resolution, coordinate reference system (EPSG:4326), and value ranges to confirm that it contains plausible elevation values across Nigeria. To ensure that elevation data align exactly with the study area, Nigeria administrative boundary vectors (ADM2) were loaded and dissolved into a single national polygon, which was then used to crop and mask the DEM so that only land area within Nigeria’s borders is retained. This preprocessing step removes unnecessary buffer regions and administrative artefacts while preserving the full spatial detail of the terrain surface. The resulting Nigeria‑only DEM will be used to derive fixed structural geographic features (e.g. mean elevation, slope, and terrain roughness), which form part of the static spatial feature set (Zi) describing long‑run physical characteristics of each grid cell.
